# Museum Map Demo

Interactive demo for visual exploration of painting clusters generated with the `museum-map` Python package.

The demo shows a similarity graph where:
- nodes are paintings,
- edges represent local visual similarity,
- colors indicate cluster membership,
- clicking a painting opens its metadata and full preview.
